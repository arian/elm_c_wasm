Elm-Wasm demo pages

Proof of concept Simplest possible app to get the integration set up between Elm and Wasm
  • Basic up/down counter (copied from Ellie default app)
  • Only the update function is C compiled to Wasm, everything else is Elm compiled to JS
  • Use Emscripten to compile C → JS + Wasm
  • Use a Bash build script to patch the Elm and Emscripten JS code together. (Something a future Elm compiler could do)
  • Use the simplest possible types for integrating with C: Model and Msg types are both Int
